# proj-final-poo-1
Projeto final da matéria de Programação Orientada a Objetos 1. Esse é um terminal simples, para Windows, com comandos simples.

## Comandos, seus usos e sintaxes:

### pwd

Exibe o diretório atual. Uso: **pwd**

### ls

Lista os arquivos e diretórios do diretório atual. Uso: **ls**

### cd

Navega entre diretórios. Uso: **cd "diretório absoluto"**

### mkdir

Cria um diretório. Uso: **mkdir "diretório, absoluto ou relativo, com ou sem subdiretórios"**

### touch

Cria um novo arquivo vazio. Uso: **touch <nome do arquivo>**

### rm

Deleta um arquivo ou diretório. Uso: **rm <arquivo/diretório, relativo>**

### cat

Abre um arquivo. Uso: **cat <arquivo, relativo>**

### echo

Escreve texto em um arquivo. Uso: **echo "texto com espaços" > arquivo**

### history

Mostra o histórico de comandos. Uso: **history**

### exit

Sai do programa. Uso: **exit**
